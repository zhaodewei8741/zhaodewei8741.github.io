数组方法
发布于：2021-01-21 19:06:05 标签：/ js / 访问：

描述

数组是一种类列表对象，它的原型中提供了遍历和修改元素的相关操作。JavaScript数组的长度和元素类型都算是非固定的。由于数组长度可变，且其数据在内存中也可以不连续（稀疏数组），所以JavaScript数组不一定是密集型的，这取决于它的使用方式。

方法

Array.from()

从类数组对象或可迭代对象中创建一个新的数组实例

若传入参数不是类数组对象或可迭代对象，则返回空数组

参数1 arrayLike — 想要转换成数组的伪数组对象或可迭代对象

参数2 [mapFn] — 若指定了该参数，新数组中的每个元素会执行该回调函数

参数3 [thisArg] — 执行回调函数mapFn时的this对象

返回值 — 一个新的数组实例

Array.isArray()

用来判断某个变量是否是一个数组对象

参数 obj — 需要检测的值

返回值 — true/false

修改器方法

这些方法会改变调用他们的对象自身的值，即直接操作对象实例

Array.prototype.pop()

删除数组最后一个元素，并返回这个元素

Array.prototype.push()

在数组末尾增加一个或多个元素，并返回数组的新长度

参数 elementN — 被添加到数组末尾的元素

Array.prototype.shift()

删除数组的第一个元素，并返回这个元素

Array.prototype.unshift()

在数组的开头增加一个或多个元素，并返回数组的新长度

参数 elementN — 被添加到数组开头的元素

pop(), push(), shift(), unshift()称为数组的栈方法

Array.prototype.reverse()

数组倒序，返回倒序后的数组

Array.prototype.splice()

数组的增删改方法，可以在任意的位置给数组添加或删除任意个元素

参数1 start — 指定的修改起始位置。若超出数组长度，则从末尾开始添加内容；若为负值，则从数组末尾开始计数；如果负值的绝对值大于数组长度，则起始位置为0

参数2 [deleteCount] — 表示要删除的数组元素的个数。若省略，或其值大于array.length - start，则start之后所有元素都会被删除。为0时， 不删除元素。

参数3 [item1,item2] — 从start位置开始添加的数组元素。若不指定，splice()只删除数组元素

返回值 — 由被删除元素组成的一个数组。未删除元素则返回空数组。

Array.prototype.fill()

用一个固定值填充一个数组中从起始索引至终止索引内的全部元素。不包括终止索引。

参数1 value — 用来填充数组元素的值

参数2 start — 起始索引，默认为0。若为负值，则从末尾开始取-start个值为起始索引，向后替换end项

参数3 end — 终止索引， 默认为数组长度

返回值 — 修改后的数组

Array.prototype.sort()

sort() 方法用原地算法对数组的元素进行排序，并返回数组。默认排序顺序是在将元素转换为字符串，然后比较它们的UTF-16代码单元值序列。由于它取决于具体实现，因此无法保证排序的时间和空间复杂性。

参数1 [callback] — 用来指定按某种顺序进行排列的函数。如果省略，元素按照转换成的字符串的各个字符的Unicode进行排序

​ element1 — 第一个用于比较的元素

​ element2 — 第二个用于比较的元素

返回值 — 排序后的数组

访问方法

这些方法不会改变调用它们的对象的值，会返回一个新数组或其他期望值

Array.prototype.concat()

返回一个由当前数组和其他若干个数组或者若干个非数组值组合成的新数组

参数 [valueN] — 数组/值，将被合并到一个新数组中。若省略，则相当于进行一次深拷贝。

Array.prototype.includes()

判断当前数组是否包含某个指定的值，返回true/false

参数1 value — 需要查找的值

参数2 [index] — 从index索引处开始查找

Array.prototype.join()

连接所有元素组成一个字符串

参数 [str] — 指定一个字符串分隔数组的每个元素。缺省时，数组元素使用,分隔

Array.prototype.slice()

截取当前数组的一部分组合成一个新数组

参数1 [start] — 起始索引（包含） 缺省则从0开始

参数2 [end] — 终止索引（不包含）缺省则截取到数组末尾

返回值 — 包含截取元素的新数组

Array.prototype.indexOf()

返回数组中第一个与指定值相等的元素的索引，找不到则返回-1

Array.prototype.lastIndexOf()

返回数组中最后一个与指定值相等的元素的索引，找不到则返回-1

迭代方法

Array.prototype.forEach()

按升序对数组的每个元素执行一次回调函数，已删除或者未初始化的项会被跳过（如稀疏数组）。

参数1 callback — 每个元素执行的函数，该函数接收1-3个参数

​ value — 数组当前项

​ [index] — 当前项的索引

​ [array] — forEach()方法正在操作的数组

参数2 [thisArg] — 执行callback时， 用作this的值

返回值 — undefined

forEach()的遍历范围在第一次调用callback前就会确定。

除抛出异常之外，没有办法中止或跳出forEach()循环。

Array.prototype.every()

如果数组中的每个元素都满足测试函数，则返回 true，否则返回 false。（与运算）

Array.prototype.some()

如果数组中至少有一个元素满足测试函数，则返回 true，否则返回 false。（或运算）

Array.prototype.filter()

将所有在过滤函数中返回 true 的数组元素放进一个新数组中并返回。

Array.prototype.find()

找到第一个满足测试函数的元素并返回那个元素的值，如果找不到，则返回 undefined。

Array.prototype.findIndex()

找到第一个满足测试函数的元素并返回那个元素的索引，如果找不到，则返回 -1。

Array.prototype.map()

返回一个由回调函数的返回值组成的新数组。

Array.prototype.reduce()

从左到右为每个数组元素执行一次回调函数，并把上次回调函数的返回值放在一个暂存器中传给下次回调函数，并返回最后一次回调函数的返回值。

参数1 callback — 回调函数

​ count — 累计器。累计器累计回调的返回值，它是上一次调用回调时返回的累积值，或者初始值

​ value — 正在处理的当前元素

​ [index] — 当前元素的索引，若提供了初始值，则起始索引为0，否则为1

​ [array] — 调用reduce()的数组

参数2 [initialValue] — 初始值。作为第一次调用callback函数时的第一个参数的值。如果没有提供初始值，则使用数组中的第一个元素。在没有初始值的空数组上调用reduce()会报错
